Contains scripts to download gridMET data (http://www.climatologylab.org/gridmet.html) at 1,106 thermoelectric plant locations within the CONUS. There are 20 thermoelectric plants outside of the CONUS (10 in Hawaii, 6 in Alaska, and 6 in Puerto Rico) which will be included on this repository soon, using data from possibly the following data sources (WRF, NOAA CFSRv2, GLDAS, MERRA-2).
